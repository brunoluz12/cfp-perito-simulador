9.1 Hash e cadeia de custódia

A função hash é um algoritmo de autenticação digital que gera, para cada arquivo, uma sequência individualizadora de caracteres, com o objetivo de garantir a integridade dos dados e a cadeia de custódia.

SHA-256 (muito usado na Polícia Federal) Função hash que transforma qualquer arquivo digital em uma sequência única e fixa de 64 caracteres hexadecimais (256 bits). Se um único bit do arquivo mudar, o hash muda completamente — por isso prova a integridade.
Truque para muitos arquivos (pasta de imagens/vídeos) Pode ser inviável colocar no laudo uma tabela com o hash de todos. Alternativa simples:
  1. Colocar todos os arquivos numa mesma pasta;
  2. Gerar um arquivo .txt com a relação dos hashes de cada um;
  3. Calcular o hash apenas desse arquivo .txt e consignar este no laudo.

Principais formatos de arquivos de imagem

FormatoSignificadoCaracterísticas
JPEG / JPG Joint Photographic Experts Group O mais comum para fotografia. Comprime bastante (perde um pouco de qualidade), gera arquivos leves. ⚠ Desvantagem: a cada vez que é salvo, há nova compressão e a qualidade diminui cumulativamente.
TIFF Tagged Image File Format Usado por profissionais para impressão e edição de alta qualidade — preserva todos os detalhes. Gera arquivos pesados.
RAW Formato Cru (bruto) Preserva os dados originais de luz e cor, sem processamento, compressão ou perda. Arquivos pesados. Importante para quem precisa da máxima qualidade para pós-processamento. Formatos proprietários (Canon, Nikon, Sony) exigem softwares específicos.
★ Macete — leveza × qualidade
JPEG = leve, mas perde qualidade ao re-salvar  ·  TIFF = pesado, máxima qualidade para edição/impressão  ·  RAW = cru, dados originais sem perda (o "negativo digital").

9.2 Dossiê

Os arquivos de um registro fotográfico não devem ser descartados. É desejável que, para cada exame, o perito crie um dossiê com todos os arquivos de imagens, arquivando-o no computador e/ou em pasta de rede.

Por que manter o dossiê?
  • Útil em caso de laudo complementar;
  • Salvaguarda do perito em caso de contestação.
